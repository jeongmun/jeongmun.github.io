Github 블로그 만든 후기

퇴사하고 시간이 좀 생기고, 앞으로 공부한 것들을 잘 정리하기 위해서 Github 블로그를 만들었다.

조금 찾아보니 jekyll와 hexo 기반으로 블로그 만드는 글이 많았다. 최근에는 hexo를 추천하는 글이 많았고, hexo 테마들이 괜찮아보였다. 그중 icarus는 테마가 깔끔하고 관련 글이 많았고, Tranquilpeak는 이쁜데 관련 글이 적었다. 그래서 icarus 테마로 시작했다.

결과부터 말하면 역시나 쉽지 않았다. hexo가 node.js를 기반인데 일단 java script, html와 css를 몰라서 블로그 보고 따라하기도 힘들었다. 그래서 블로그 만들고 이 후기 적는데 꽤 오래 걸렸다.

이제 반 했으니 블로그를 만들었으니 틈틈이 글을 올려볼 생각이다.
